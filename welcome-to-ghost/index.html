Can I get an AMEN for Google Chrome's Dev tools!

23 April 2015 on Getting Started

Especially Google’s Javascript V8 engine. The command line tool is magnifcique(sp?)! However, over time I’ve found that “CMD+TAB”ing between sublime and chrome to be a bit annoying.

[enter sublime-v8 plugin]



I’d like to thank Akira-cn for making a Sublime plug-in that allows us to run Google's V8 in Sublime. Now I can use the command line inside Sublime.

Installation Steps:
  1. Download Sublime Text
  2. Download Package Control
  3. Open SublimeText and type the following keys CMD+Shift+P
  4. A small box will appear, inside of which type the following "Package Control: Install Package"

  1. Click/Select "Package Control: Install Package" 6. Type and select the following into the next small box that appears "sublime-v8"
  2. Once downloaded, restart SublimeText
  3. CTRL+ALT+J key your way to the console!
